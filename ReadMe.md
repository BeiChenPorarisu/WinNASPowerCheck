# 停电监控与自动休眠程序

## 简介
本程序通过定期 ping 路由器 IP 检测网络通断，当发生停电导致断网时，可自动执行系统休眠或关机，防止数据丢失或硬件损坏。提供图形化界面与系统托盘图标，支持多种灵活配置。

## 功能特点
- 定时检测网络状态（ping 路由器）
- 断网后先进入静默期（不弹窗），快速探测恢复
- 静默期结束后弹出警告窗口，显示倒计时，可暂停/重置/立即执行操作
- 支持休眠、关机两种操作方式
- 系统托盘图标动态变化：正常（灰色）、静默期（黄色闪烁）、警告期（红色闪烁）
- 深色/浅色主题切换，自定义字体
- 所有设置通过图形界面修改，自动保存配置文件 `config.json`

## 系统要求
- 操作系统：Windows 10/11（其他系统未测试）
- Python 环境（如运行源代码）：
  - Python 3.6+
  - 依赖库：`pystray`、`pillow`（托盘图标功能）和 `tkinter`（通常内置）

## 使用说明
1. 程序启动后仅在系统托盘显示灰色电源图标，无主窗口。
2. **右键托盘图标**可打开菜单：“显示窗口”和“退出”。
3. **断网时**：
   - 进入静默期（默认30秒），托盘图标黄色闪烁，无弹窗。
   - 静默期结束后若网络未恢复，弹窗并开始倒计时（默认3分钟），托盘图标红色闪烁。
   - 弹窗中可暂停/重置倒计时，或点击“立即休眠”/“立即关机”直接执行操作。
   - 关闭窗口则退出停电状态（不执行操作）。
4. **网络恢复**：窗口自动隐藏，托盘图标恢复灰色。
5. **设置**：点击窗口中的“设置”按钮可修改所有参数（IP、ping间隔、倒计时时长、静默期、操作类型、字体、主题等）。部分参数需重启生效。

## 配置项说明
配置文件 `config.json`（与程序同目录）可通过设置窗口修改，主要字段：
| 字段 | 说明 | 默认值 |
|------|------|--------|
| `target_ip` | 路由器 IP | 192.168.1.1 |
| `ping_interval` | 正常 ping 间隔（秒） | 60 |
| `probe_ping_interval` | 静默期 ping 间隔（秒） | 5 |
| `alert_ping_interval` | 警告期 ping 间隔（秒） | 1 |
| `countdown_seconds` | 倒计时总时长（秒） | 180 |
| `probe_seconds` | 静默期时长（秒） | 30 |
| `power_action` | 操作类型（hibernate/shutdown） | hibernate |
| `font_family` | 界面字体 | Microsoft YaHei |
| `dark_mode` | 深色主题开关 | true |

## 注意事项
- 休眠/关机需要系统权限，建议以管理员身份运行程序。
- 托盘图标依赖 `pystray` 和 `pillow`，如无法显示请检查依赖安装或尝试以管理员身份运行。
